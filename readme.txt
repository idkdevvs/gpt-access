
Bienvenue dans le jeu "Echappez-vous de la Terre" ! Ce jeu est un récit d'aventure dans lequel vous incarnez un extraterrestre capturé par des scientifiques terriens. Pour vous échapper, vous devez créer du code informatique et récupérer quatre morceaux d'une carte qui vous permettront de rentrer chez vous à bord de votre vaisseau spatial. Ces fragments sont dispersés dans quatre des neuf niveaux que vous devrez traverser pour compléter le jeu. Ainsi, pour achever votre quête, vous devrez collecter les quatre fragments et les utiliser pour résoudre le neuvième niveau.

Les bibliothèques suivantes sont incluses dans le code. Voici les commandes pour les télécharger:

pyqt6 : pip install pyqt6
pillow : pip install pillow

shutil : inclus dans la bibliothèque standard de Python
functools : inclus dans la bibliothèque standard de Python
json : inclus dans la bibliothèque standard de Python
Elles sont nécessaire au bon fonctionnement du jeu.

Notez que les réponses sont stockées dans un fichier séparé nommé "solution_07.txt". C'est dans ce fichier que vous trouverez les réponses aux énigmes.

Pour terminer le jeu, vous devez :

Compléter tous les neuf niveaux en collectant les fragments de carte dans quatre des niveaux.
Aller dans les images débloquées pour voir les images.
Télécharger les quatre images une fois que les quatre fragments sont débloqués.
Assembler les images pour résoudre le neuvième niveau et terminer le jeu.
